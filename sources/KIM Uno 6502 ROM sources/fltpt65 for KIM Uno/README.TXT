CBA65 NOTES
===========

cbA65 is a DOS application which will run under Windows XP or Vista.
It is a full-featured assembler intended for general 65xx program
development.

Make sure you have a suitable folder which is in the search path
for your DOS applications. Unzip the cba65.zip archive into this
folder.

No other steps are required to activate or install the assembler.

The files included are:

- cbA65.exe     The executable assembler
- cbA65.pdf     A User's Guide and Reference Manual
- readme.txt    This file
